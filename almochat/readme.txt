AI-powered Chatbot solution that helps businesses automate customer interactions to manage issue resolutions, feedback collection, and more.

== Description ==
 Almo is an AI-powered custom chatbot that is trained on your content and knowledge base (Website, Documents, Text, Q&A, Existing Customer Support Tickets) and is seamlessly embedded on your Website.

Once embedded on the website, Almo acts as a 24x7 live Customer Support and helps businesses enhance customer interactions.

- Realtime access - Almo Chat is trained on and has access to the entire knowledge base in real-time.

- Instant Response - Almo Chat can generate instant responses for queries and conversations.


== Installation ==
This section describes how to install the plugin and get it working.
1. Upload [`almochat`](https://github.com/lekhakai/almochat_wp_plugin/archive/refs/heads/main.zip) to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Go to Almo Chat plugin menu and save the chatbot ID.

4. You are good to Go.

== Frequently Asked Questions ==

= How can I get a chatbot ID? =

1. Login to your Almo chat Account [AlmoChat](https://www.almochat.com)
2. If you create an account it does not exist yet.
3. Create a chatbot following the steps written there.
4. After the chatbot is created, go to the dashboard, select your chatbot and copy the chatbot ID of the chatbot you want to use.

= Can I use a chatbot without a chatbot ID? =
No, If the chatbot ID is not added. Users can not use Chatbot. If the chatbot ID is incorrect It will show Error 404 Page instead of chatbot.
